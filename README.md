# Memory-Game-

A browser-based memory card matching game built with HTML, CSS, and JavaScript. Players flip cards to find matching image pairs across multiple difficulty levels.

---

## Features

- 3 difficulty levels:
  - 4 Cards
  - 8 Cards
  - 12 Cards
- Card flip animations
- Match checking system
- Win animation
- Sidebar level selector
- Custom images for each level
- “How to Play” popup menu
- Responsive game layout

---

## How to Play

1. Choose a difficulty level from the sidebar
2. Click on a card to flip it
3. Flip a second card to find a matching pair
4. If the cards match, they stay flipped
5. If they do not match, they flip back over
6. Match all cards to win the game
